## üöÄ Getting Started

### Prerequisites
*   [Node.js](https://nodejs.org/) (v18 or higher)
*   npm or yarn

### Installation
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/AadityasinhJadeja/Countdown-2026.git
    cd Countdown-2026
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Configure Environment:**
    Copy the `.env.example` to `.env.local` and add your Gemini API key (optional for the prediction engine).
    ```bash
    cp .env.example .env.local
    ```

4.  **Run locally:**
    ```bash
    npm run dev
    ```

## üåê Deployment

The project is optimized for deployment on **Vercel** or **GitHub Pages**. Ensure your `GEMINI_API_KEY` is added to your deployment platform's environment variables.
